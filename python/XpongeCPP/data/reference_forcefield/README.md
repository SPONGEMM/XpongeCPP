# Reference Xponge Force-Field Snapshot

This directory is a read-only data snapshot copied from the reference Xponge
repository path `Xponge/forcefield`.

Amber data is also mirrored under `XpongeCPP/data/amber` for the active v1 C++
loading path. Other force-field families are packaged here so later C++ parsers
can be implemented without depending on the original Xponge source checkout.
