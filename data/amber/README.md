# Amber Data

Amber-compatible template and parameter data used by XpongeCPP is packaged under
`python/XpongeCPP/data/amber` so it is available from installed wheels via
`importlib.resources`.

Those files are copied from the reference Xponge repository path
`Xponge/forcefield/amber` and include ff14SB, ff19SB, TIP3P/TIP4P/SPCE/OPC,
GAFF/GAFF2, ions, lipid, and GLYCAM data present in that source tree.
